# Entregable 3

&nbsp;

## Correr el proyecto:

###

-   git clone https://github.com/Matias8776/Entregable3.git
-   cd Entregable3
-   npm install
-   npm start

## Mostrar productos:

###

-   localhost:8080/api/products

## Buscar producto por ID:

###

-   localhost:8080/api/products/1

## Mostrar productos con limite:

###

-   localhost:8080/api/products?limit=5
